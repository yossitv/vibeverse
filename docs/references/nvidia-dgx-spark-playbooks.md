# NVIDIA DGX Spark Playbooks - GitHub Reference

Source: https://github.com/NVIDIA/dgx-spark-playbooks

## Overview

A collection of **42 step-by-step playbooks** for setting up AI/ML workloads on NVIDIA DGX Spark devices with Blackwell architecture.

**Stats:** 582 stars, 152 forks, 241 commits
**License:** Apache 2.0
**Languages:** Jupyter Notebook (69.3%), TypeScript (18.8%), Python (10.1%)
